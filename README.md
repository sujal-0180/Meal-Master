# Meal Master – Recipe Finder App

# Project Overview

Meal Master is a web application that helps users discover recipes based on ingredients they already have. Instead of wondering "what to cook," users can simply enter ingredients and get a list of meals they can prepare.

This project focuses on improving everyday cooking decisions while demonstrating modern web development skills using React and APIs.

# Features

*  Search recipes by ingredient
*  View detailed recipe information
*  Display ingredients and cooking instructions
*  Show meal images for better visualization
*  Fast and responsive user interface
*  Dynamic data fetching from API


# API Used

https://www.themealdb.com/api.php

The API provides:

* Meal search by ingredient
* Recipe details
* Images and instructions
